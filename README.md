# Installation

```shell
sls create --name <SERVICE_NAME> --template-url https://github.com/falled10/python-serverless-template
```

```shell
cd <SERVICE_NAME>
```

```shell
python3 -m venv venv
```

```shell
source venv/bin/activate
```

```shell
pip install -r requirements.txt
```

```shell
npm install
```
